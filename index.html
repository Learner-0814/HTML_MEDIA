A Group 2 Presentation

This is to make the website more dynamic and user-friendly by using HTML.


Images

In adding elements like images, developers need to input the exact file path and file format (.JPG, .png, .jpg) of the specific image.
example code

<!DOCTYPE>
<body>
    <meta charset="UTF-8">
 <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
<img src="filename.JPG" alt="My Photo" width="500">



Video

    <source src="videos/video.mp4" type="video/mp4">



Audio

This adds an audio file to the website. .MP3 works best for HTML

Example code


 <audio controls>
 <source src="audio/song.mp3" type="audio/mp3"> </audio>
